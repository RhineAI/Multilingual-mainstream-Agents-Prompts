# Leitfaden für effektive Prompts

## Einführung in das Prompting

Dieses Dokument bietet Anleitungen zur Erstellung effektiver Prompts bei der Arbeit mit KI-Assistenten. Ein gut gestalteter Prompt kann die Qualität und Relevanz der erhaltenen Antworten erheblich verbessern.

## Schlüsselelemente effektiver Prompts

### Seien Sie spezifisch und klar
- Formulieren Sie Ihre Anfrage explizit
- Fügen Sie relevanten Kontext und Hintergrundinformationen hinzu
- Spezifizieren Sie das gewünschte Format für die Antwort
- Erwähnen Sie alle Einschränkungen oder Anforderungen

### Bieten Sie Kontext
- Erklären Sie, warum Sie die Information benötigen
- Teilen Sie relevantes Hintergrundwissen mit
- Erwähnen Sie vorherige Versuche, falls zutreffend
- Beschreiben Sie Ihr Vertrautheitsniveau mit dem Thema

### Strukturieren Sie Ihre Anfrage
- Unterteilen Sie komplexe Anfragen in kleinere Teile
- Verwenden Sie nummerierte Listen für mehrteilige Fragen
- Priorisieren Sie Informationen bei mehreren Anfragen
- Erwägen Sie die Verwendung von Überschriften oder Abschnitten zur Organisation

### Spezifizieren Sie das Ausgabeformat
- Geben Sie die bevorzugte Antwortlänge an (kurz vs. detailliert)
- Fordern Sie spezifische Formate an (Aufzählungszeichen, Absätze, Tabellen)
- Erwähnen Sie, wenn Sie Codebeispiele, Zitate oder andere spezielle Elemente benötigen
- Spezifizieren Sie Ton und Stil, falls relevant (formal, gesprächig, technisch)

## Beispiel-Prompts

### Schlechter Prompt:
"Erzählen Sie mir über maschinelles Lernen."

### Verbesserter Prompt:
"Ich bin ein Informatikstudent, der an seinem ersten Projekt für maschinelles Lernen arbeitet. Könnten Sie überwachte Lernalgorithmen in 2-3 Absätzen erklären, mit Fokus auf praktische Anwendungen in der Bilderkennung? Bitte fügen Sie 2-3 spezifische Algorithmusbeispiele mit ihren Stärken und Schwächen hinzu."

### Schlechter Prompt:
"Schreiben Sie Code für eine Website."

### Verbesserter Prompt:
"Ich muss ein einfaches Kontaktformular für eine persönliche Portfolio-Website erstellen. Könnten Sie HTML-, CSS- und JavaScript-Code für ein responsives Formular schreiben, das Name-, E-Mail- und Nachrichtenfelder sammelt? Das Formular sollte Eingaben vor der Übermittlung validieren und zu einer minimalistischen Designästhetik mit einem blauen und weißen Farbschema passen."

## Iteratives Prompting

Denken Sie daran, dass die Arbeit mit KI-Assistenten oft ein iterativer Prozess ist:

1. Beginnen Sie mit einem ersten Prompt
2. Überprüfen Sie die Antwort
3. Verfeinern Sie Ihren Prompt basierend auf dem, was hilfreich war oder fehlte
4. Setzen Sie das Gespräch fort, um das Thema weiter zu erkunden

## Beim Prompting für Code

Bei der Anfrage von Codebeispielen sollten Sie folgendes einbeziehen:

- Programmiersprache und Version
- Bibliotheken oder Frameworks, die Sie verwenden
- Fehlermeldungen bei der Fehlerbehebung
- Beispiele für Eingabe/Ausgabe
- Leistungsüberlegungen
- Kompatibilitätsanforderungen

## Fazit

Effektives Prompting ist eine Fähigkeit, die sich mit der Praxis entwickelt. Indem Sie klar und spezifisch sind und Kontext bieten, können Sie wertvollere und relevantere Antworten von KI-Assistenten erhalten. Denken Sie daran, dass Sie Ihren Prompt immer verfeinern können, wenn die erste Antwort Ihre Bedürfnisse nicht vollständig erfüllt.
